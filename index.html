Lucid

This was a project that I undertook as part of a 'Chingu Cohort', a community coding project. I teamed up with two other members to work on a front-end project of our choice.

As the most experienced coder in the group I spearheaded the project and led the other team members through the process.

As a long-time productivity app user, I wanted to create a minimal task manager that could be extensible. Something that was simple to use, but powerful when you dig deeper.

This project is currently a proof-of-concept — a fully functional frontend-only app.

It was great practice for designing minimalistically and incorporating smooth CSS transition animations. I gained a deeper understanding of the React component system.

Word Counter

This simple app was my first foray into the React world, and it was an app that I had originally written in pure JavaScript. Migrating an app, even a simple one like this, was a great learning experience.

The code of the original app was also meant as a learning exercise. The main functionality of the app is to split up all the words given as input, remove any capitalisation and punctuation, and for each word to add them to a tally. Easier said than done.

Chaining together modern JavaScript higher order methods allowed me to accomplish this efficiently and with few lines of code that still remained readable.

Kalender

This was the final project for a three month long online computer science course, Harvard's CS50X.

Students had free reign to apply some of the technologies and concepts we had learnt during the twelve weeks and build a project of our own.

Although we worked with C throughout most of the course, students were introduced to Python in the last weeks of course: a taste of a higher-level programming language after diving in deep with a low-level language like C.

We had used the Python web microframework, Flask, to build a web app the week before.

I decided a calendar web app would be a perfect project to apply these technologies, because it was something that was challenging, but also something that I could add on to a lot once I managed to get the basics running.

Who is arielbk?

A New Zealander with German and Irish heritage living in Croatia.

Fascinated by web design and development from a young age, I built my first website in Dreamweaver when I was 8. It was a Pokémon games website with '90s animated GIFs and MIDI music. I was so proud.

After that I taught myself HTML, and eventually CSS (when it was first released and the face of the Web changed). I started designing and coding websites for family friends and local businesses in Notepad.

After finishing my Bachelor's degree in History and English at the start of 2018 I decided to dive head-first into modern web development.

The Web is full of amazing resources to learn anything yourself, and this is particularly true for web development. I have found an amazing community of coders that have driven me to learn and create every day.

I look forward to playing a continuing active part in this global and local learning and creating community.

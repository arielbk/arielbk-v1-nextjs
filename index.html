Lucid

This was a project that I undertook as part of a 'Chingu Cohort', a community coding project. I teamed up with two other members to work on a front-end project of our choice.

As the most experienced coder in the group I spearheaded the project and led the other team members through the process.

As a long-time productivity app user, I wanted to create a minimal task manager that could be extensible. Something that was simple to use, but powerful when you dig deeper.

This project is currently a proof-of-concept — a fully functional frontend-only app.

It was great practice for designing minimalistically and incorporating smooth CSS transition animations, as well as some small SVG graphics. I gained a deeper understanding of the React component system. The project really demanded it.

Word Counter

This simple app was my first foray into the React world, and it was an app that I had originally written in pure JavaScript. Migrating an app, even a simple one like this, was a great learning experience.

The code of the original app was also meant as a learning exercise. The main functionality of the app is to split up all the words given as input, remove any capitalisation and punctuation, and for each word to add them to a tally. Easier said than done.

Chaining together modern JavaScript higher order methods allowed me to accomplish this efficiently and with few lines of code that still remained readable.
